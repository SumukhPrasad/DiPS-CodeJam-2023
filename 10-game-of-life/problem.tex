	\section*{Prompt}
	Conway's Game of Life is a zero-player game defined as a grid of cells, each having a state that corresponds to either ``dead'' or ``alive.'' The cells follow the following rules:
	
	\begin{itemize}
		\item Any live cell with two or three live neighbours survives.
		\item Any dead cell with three live neighbours becomes a live cell.
		\item All other live cells die in the next generation. Similarly, all other dead cells stay dead.
	\end{itemize}
	
	Your task, given a starting condition for the game, is to compute the next generation of cells and find how many cells are alive.


	\subsection*{Input Format}
	\begin{itemize}
		\item The first line contains an integer $n$, denoting the size of the grid.
		\item The next $n$ line contain $n$ space-separated values each, either $0$ or $1$.
	\end{itemize}

	\subsection*{Output Format}
	Your output must contain a single integer denoting how many cells are alive in the next generation.
