	\section*{Prompt}
	Given an array of $n$ integers, determine if dividing the list into groups of increasing and decreasing elements yields a list of equal-sized lists. In other words, determine if the ``turning points'' of the list are spaced out evenly, i.e., the list oscillates between increasing and decreasing ``periodically''. If the an element exists that is neither increasing nor decreasing, assume that the list does not oscillate.

	\subsection*{Input Format}
	The first and only line of input contains a space-separated list of integers.

	\subsection*{Output Format}
	Your output must contain ``true'' if the list oscillates periodically, and ``false'' if it cannot.

	\subsection*{Sample Input/Output}

		\hline

	\section*{Solution}
	Taking the example $n[0, 3, 7, 5, 2, 3, 6]$:
	\begin{itemize}
		\item Compute the deltas: $n_{\Delta}[3, 4, -2, -3, 1, 3]$
		\item Compute the sign of the deltas: $n_{\text{sgn }\Delta}[1, 1, -1, -1, 1, 1]$
		\item Group by equality: $n_{\text{sgn }\Delta\text{ grouped}}[[1, 1], [-1, -1], [1, 1]]$
		\item Find the lengths of the groups: $n_{\text{length of sgn }\Delta{ grouped}}[2, 2, 2]$
		\item If $len(\{n: n\in n[]\}) = 1$, then the list matches the condition. 
	\end{itemize}
