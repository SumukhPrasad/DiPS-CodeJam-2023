	\section*{Prompt}
	Given a string, determine if the string has two or more consecutive substrings that are equivalent. For example, given the strings $s_1[\text{bdefdefg}]$ and $s_2[\text{abcab}]$, $s_1$ will return true as the substring ``def'' repeats, while $s_2$ returns false.


	\subsection*{Input Format}
	The first and only line of input contains string $s$.
	
	\subsection*{Output Format}
	Print ``true'' if the string has two or more consecutive substrings that are equivalent, and print ``false'' otherwise.

	\subsection*{Sample Input/Output}
	\begin{tabular}{ |l|l| } 
		\hline
		\textbf{Input} & \textbf{Output} \\
		{\texttt{bdefdefg}} & {\texttt{true}} \\ % use {\lstinputlisting{./testCases/input/input00.txt}} & {\lstinputlisting{./testCases/output/output00.txt}}
		\hline
	\end{tabular}


	\section*{Solution}
	\begin{itemize}
		\item Find all substrings of the string.
		\item For each substring:
		\begin{itemize}
			\item Check if double the substring (``abc'' > ``abcabc'') is also present in the list of substrings.
			\item If such a string is present, print ``true'' and exit the loop.
		\end{itemize}
		\item If the loop has run through and no repeating substrings have been found, print ``false.''
	\end{itemize}
